## Следующие шаги для интеграции

1. **Клонировать GitHub репозиторий:**
   ```bash
   git clone https://github.com/suetin/dino-game.git
   cd dino-game
   ```

2. **Установить зависимости стартового скелета:**
   ```bash
   yarn bootstrap
   ```

3. **Переместить код:**
   - Скопировать содержимое `frontend/` → `packages/client/`
   - Скопировать содержимое `backend/` → `packages/server/`

4. **Проверить работу:**
   ```bash
   yarn bootstrap  # Переустановить зависимости
   yarn dev --scope=client
   yarn dev --scope=server
   ```

5. **Следовать MIGRATION_GUIDE.md** для детальных инструкций
