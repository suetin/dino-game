# Перенос и интеграция (Migration Guide)

## Цель

Руководство по переносу кода, данных и настройке окружения при интеграции с существующими системами или при переезде на новый репозиторий.

## Рекомендации

1. **Окружение** — использовать переменные из `.env.sample`, не коммитить секреты.
2. **База данных** — миграции и сиды описать в `packages/server`; при переносе БД — бэкап и восстановление по инструкции проекта.
3. **Клиент/сервер** — после клонирования выполнить `yarn bootstrap`, затем запуск по [SETUP_GUIDE.md](SETUP_GUIDE.md).

Детали по архитектуре: [ARCHITECTURE.md](ARCHITECTURE.md).
